# Story Saver

Story Saver is a browser-based tool for downloading public Instagram stories, reels, videos, photos, and highlights from direct links.

## Website
https://savestory.online/

## What it does
- Save public Instagram stories
- Download reels from direct links
- Save public videos and photos
- Download public highlights
- Works in desktop and mobile browsers

## How it works
1. Copy a public Instagram link
2. Paste it into Story Saver
3. Download the available media

## Notes
Story Saver is designed for public content only and does not bypass private or restricted media.
